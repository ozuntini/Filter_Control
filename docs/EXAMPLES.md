# 🚀 Exemples d'utilisation
## Test d'utilisation de la bibliothèque
Vous pouvez utiliser le script bash FC_Launche.sh pour tester l'utilisation de la bibliothèque. Assurez-vous d'avoir les permissions nécessaires pour exécuter le script.

```bash
./FC_Launche.sh
=== Lanceur test_filter_controller ===
usage: ./FC_Launch.sh [-o|--open ou -c|--close ou -s|--status]
```
Avec les différentes options disponibles vous pouvez ouvrir, fermer ou vérifier le statut du contrôleur de filtre.
```bash
./FC_Launch.sh -o # Ouvre le contrôleur de filtre
./FC_Launch.sh -c # Ferme le contrôleur de filtre
./FC_Launch.sh -s # Vérifie le statut du contrôleur de filtre
```
